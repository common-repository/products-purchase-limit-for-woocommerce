A plugin to set minimum and maximum products purchase limits for WooCommerce Shop.

== Description ==

This plugin allows you to set up minimum or maximum purchase limits for products in your shop. You can either set minimum or maximum products purchase limit or can set both minimum and maximum products purchase limit. Purchases must meet these conditions to proceed to checkout. It manage your customers to buy a certain amount of products, or stop them from buying too much.

This plugin first checks the total number of products on cart, if a purchase in your shop fails a product purchase limits, an error message displays  at the top of the cart/checkout page. Until the products purchase limits you have set have been satisfied, the customer wonâ€™t be able to checkout.


How to install?

1. Unzip the downloaded zip folder.
2. Upload the folder to '/wp-content/plugins' directory of your WordPress installation.
3. Activate the plugin via the WordPress Plugins page.
4. A new section will be automatically created on products tab on WooCommerce->Settings->Products>Products Purchase Limit

== Frequently Asked Questions ==

= Is that plugin set the limits for category based products or for user role based? =

No, it simply checks the total number of products on cart.

= Is that plugin ready for multi-site use? =

Not tested for multisite.
